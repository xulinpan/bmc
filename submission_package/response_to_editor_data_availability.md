# Response to Editorial Data-Availability Queries

Manuscript title: **A Leakage-Controlled Cold-Start Benchmark for Drug-Combination Synergy Classification on DrugComb v1.5**

Dear Editor,

Thank you for pointing out the data-availability inconsistencies and the inaccessible web link. We have revised the manuscript and submission materials as follows.

## Comment 1: Data availability statements mismatch

**Comment:** Please make sure that you are providing the same data availability statement in the manuscript and on the submission system.

**Response:** We agree and have corrected this. The manuscript Availability of data and materials section and the submission-system data-availability text now use the same statement, with the same source-data and repository links. We also added `SUBMISSION_SYSTEM_DATA_AVAILABILITY.txt` to the submission package so the exact text can be pasted into the journal system.

## Comment 2: Inaccessible web link

**Comment:** The web link previously provided in the Availability of data and materials/Data availability section did not yield search results. Please ensure that all web links provided are accessible.

**Response:** We agree and have removed the inaccessible portal link from the manuscript and submission materials. The revised statement now cites only Zenodo record 15235991 for the public DrugComb v1.5 source summary file and the public GitHub repository for the derived modeling table, code, metric outputs, and BMC submission materials:

- Source data: https://zenodo.org/records/15235991
- Submitted package/code/data: https://github.com/xulinpan/bmc

We also scanned the repository to confirm that the inaccessible portal link and Outlook safelink are no longer present.
