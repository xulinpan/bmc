The DrugComb v1.5 source summary file used in this study is publicly available
from Zenodo record 15235991 (https://zenodo.org/records/15235991). Because the
DrugComb web portal was not reliably accessible at the time of submission, the
derived modeling dataset, preprocessing and analysis code, metric outputs and
BMC submission materials are deposited in the public GitHub repository
https://github.com/xulinpan/bmc. The derived modeling dataset is provided in
this data package as data/drugcomb_synergy_prediction_modeling.csv, together
with a data dictionary, dataset profile, checksums and the preprocessing script
used to derive the table from the public DrugComb summary file.

No restricted, patient-identifiable or newly collected human participant data are
included in this package.
