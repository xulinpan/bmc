# DrugComb v1.5 Synergy Prediction Modeling Dataset

This package accompanies the BMC Bioinformatics manuscript:

**A Leakage-Controlled Cold-Start Benchmark for Drug-Combination Synergy Classification on DrugComb v1.5**

Author: Xulin Pan

## Contents

The compressed archive generated from this folder contains:

- `data/drugcomb_synergy_prediction_modeling.csv`
  - The exact derived modeling table used for the manuscript.
  - Rows: `739,964`
  - Columns: `19`
  - File size: `912,306,648` bytes
- `metadata/DATA_DICTIONARY.md`
  - Column definitions and modeling notes.
- `metadata/dataset_profile.txt`
  - Row counts, split counts, class balance, study/tissue frequencies and missingness.
- `metadata/checksums.tsv`
  - MD5 and SHA256 checksums for the modeling table and source DrugComb file.
- `scripts/prepare_modeling_dataset.py`
  - Script used to derive the modeling table from the public DrugComb v1.5 summary file.

## Source Data

The source data are public DrugComb v1.5:

- DrugComb portal: https://drugcomb.org/download/
- Zenodo record: https://zenodo.org/records/15235991
- Source file: `summary_v_1_5.csv`
- Source MD5: `cd3e24758c10ffaea0124b6e80626c32`

The modeling table is a cleaned and filtered derivative of the public source file. It contains true
drug-combination rows and excludes single-agent rows.

## Recommended Use

For the manuscript's primary task:

- Regression target: `synergy_zip`
- Classification target: `zip_synergy_label`
- Split column: `split`

The paper's main benchmark uses `drugcomb_synergy_prediction_modeling.csv` with the predefined
`train`, `validation` and `test` values in `split`. Additional cold-start splits are generated by the
analysis scripts in the manuscript submission package.

## Label Definition

ZIP synergy class labels are generated with common thresholds:

- `synergistic`: ZIP score >= 10
- `antagonistic`: ZIP score <= -10
- `additive`: otherwise

Equivalent labels are also provided for Loewe, HSA and Bliss scores.

## Leakage-Control Note

The modeling table includes synergy scores and labels for reproducibility and downstream evaluation.
When reproducing the manuscript's pre-treatment descriptor benchmark, do **not** use response-derived
fields such as `synergy_loewe`, `synergy_hsa`, `synergy_bliss`, CSS, RI, IC50 or S-score fields as
input features. The submitted benchmark uses only compound identity, cell context, clinical phase and
target-annotation descriptors.
